ℹ️ Información del Sistema

ARB-BOT es un chatbot institucional que responde preguntas basándose en documentos como el manual de convivencia, reglamentos y políticas de la institución.

¿Cómo funciona?

  1. Sube documentos institucionales (PDF, DOCX, TXT)
  2. El sistema procesa y analiza los documentos
  3. Cuando alguien pregunta, busca información relevante
  4. Genera respuestas basadas en los documentos cargados

Tecnologías utilizadas (100% gratuitas):

  • Backend: Python + Flask
  • IA: Hugging Face Transformers
  • Base de Datos: SQLite
  • Frontend: Bootstrap 5 + JavaScript
  • Modelos: DistilGPT-2, GPT-2
  • Licencia: Open Source
