## 🚂 Railway - Detalles

### Plan Gratuito
- **Primeros 30 días:** $5 crédito gratis
- **Después:** $1/mes crédito no acumulable
- **RAM:** 0.5GB por servicio
- **Ventajas:**
  - ✅ No se suspende por inactividad
  - ✅ Auto-deploy desde GitHub
  - ✅ HTTPS automático
  - ✅ Logs en tiempo real
  - ✅ Muy fácil de usar

### Plan Hobby ($5/mes)
- **RAM:** Hasta 8GB por servicio
- **Crédito:** $5/mes incluido
- **Ideal si:** Necesitas más de 0.5GB RAM

### ⚠️ Limitación para tu app
- **0.5GB puede ser insuficiente** para Torch + RAG
- **Solución:** Usa las optimizaciones ya implementadas (límite de 50 páginas)
- **O:** Plan Hobby ($5/mes) si necesitas más RAM

---

## 🎨 Render - Detalles

### Plan Gratuito
- **RAM:** 512MB
- **Suspensión:** Se apaga después de 15 min sin uso
- **Ventajas:**
  - ✅ Gratis permanentemente
  - ✅ Auto-deploy desde GitHub
  - ✅ HTTPS automático

### Plan Starter ($9/mes)
- **RAM:** 512MB (igual que gratis) ⚠️
- **CPU:** 0.5 vCPU (mejor que gratis)
- **No se suspende**

### Plan Standard ($25/mes)
- **RAM:** 2GB
- **CPU:** 1 vCPU
- **Ideal para:** Producción

### ⚠️ Problemas para tu caso
- Plan gratuito se suspende (mala experiencia)
- Starter ($9/mes) tiene misma RAM que gratis
- Standard ($25/mes) es caro para solo demo

---

## 🐳 DigitalOcean - Detalles

### Con Crédito $200
- **Duración:** 60 días crédito + ~33 meses si eliges 1GB
- **RAM:** 1GB ($6/mes) o 2GB ($12/mes)
- **Ventajas:**
  - ✅ Máximo tiempo gratis
  - ✅ Control total
  - ✅ No se suspende

### Desventajas
- ⚙️ Requiere configuración manual (Docker, Nginx)
- ⚙️ Más tiempo de setup
- ⚙️ Sin auto-deploy (se puede configurar)
